# Phase 0: Platform Foundation

## Goal

Establish the monorepo, deployment model, and shared code foundation.

## Deliverables

* Monorepo workspace structure
* Next.js dashboard package
* API service package
* Worker service package
* Shared type and utility packages
* cPanel-compatible deployment notes
* Nginx reverse proxy configuration

## Notes

This phase avoids Docker and prepares the project for direct VPS deployment.
